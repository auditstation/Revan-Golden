Odoo 16 Mastercard Payment Gateway Service (MPGS) (Enterprise)

Mastercard Payment Gateway Service (MPGS) Features for Enterprise Version

Note: This module is tested and working in Odoo Enterprise edition but not tested in Community version. And we are supporting Ubuntu OS and not Windows OS.

Features

  • Payment Processing: A single provider for all your payment needs. Your customer can pay anywhere, anytime, and through multiple channels
  • International Processing: Unlock new markets and benefit from the rapid growth of international sales
  • Fraud & Risk Management: Reduce the costs of fraud to your business whilst ensuring that genuine customers are not rejected or left frustrated

External Dependencies

  • To install this module required to install bellow library:
  • pip install pycountry

How to use

  • Website > Configuration > Payment Transaction
  • From the list of payment provider select Master Card Gateway


  • MPGS Credentials: Take Merchant ID, MPGS Secret Key and Merchant Name and Address from your MPGS account and put into MPGS Payment Provider in Odoo.


  • Company currency and API currency should be same
  • Here need to configure Journal


  • If you want to generate invoice at the time of online payment then you have to configure this settings


  • Now let's do payment from website so open website and make order
  • Select MPGS from payment methods.


  • You need to follow this payment step






  • User can see confirm order detail here


  • Transaction Record has been created in Payment Transaction


  • Here you can check sale order and invoice inside transaction


  • Here you can check sale order and invoice in details
